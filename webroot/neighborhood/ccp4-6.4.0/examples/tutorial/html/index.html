CCP4 Tutorial: Contents

How to run these tutorials

These tutorials are distributed in the directory $CCP4/examples/tutorial, in which you will find the following sub-directories:

data
Data files with which to run the tutorials
html
Instructions on running the tutorials
results
Results files from the tutorials

This file is html/index.html which you should have loaded into your browser. Choose from the tutorials below, and follow the instructions. You should start with the first tutorial which sets up some parameters necessary for later tutorials.

These tutorials were developed for release 5.0 of the CCP4 suite. They should still form a good introduction to later versions of the CCP4 suite, though be aware that some things may have changed. The CCP4 GUI "ccp4i" is used to run the programs.
