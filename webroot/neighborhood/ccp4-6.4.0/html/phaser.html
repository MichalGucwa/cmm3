PHASER (CCP4: Supported Program)

NAME

phaser-2.5.5 - Maximum Likelihood Analysis and Phasing.

SYNOPSIS

phaser

DESCRIPTION

Molecular Replacement, SAD Phasing, Anisotropy Correction, Cell Content Analysis, Translational NCS/Twin Analysis, and SCEDS Substructure Determination/Normal Mode Perturbabion of PDB files (Model Generation for Molecular Replacement).

There are some changes between this version and previous versions so input scripts may need editing.
