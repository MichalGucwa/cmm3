ZEROED (CCP4: Unsupported Program)

NAME

zeroed - Set parts of map to zero

SYNOPSIS

zeroed mapin foo_in.map mapout foo_out.map
[Input]

DESCRIPTION

The program sets specified regions of the input map to zero. Used for omit maps. See example below for interactive use.

INPUT AND OUTPUT FILES

MAPIN Input map

MAPOUT Output map

AUTHORS

Originator : Eleanor Dodson, York University

Contact : this program is unsupported

EXAMPLES


 Sample Unix input:

zeroed MAPIN 3fo2fc.map MAPOUT out.map

  INPUT LIMITS X1, X2, Y1, Y2, Z1, Z2 AND FRINGE IN AS
2 5 2 5 2 5 1
