Sample plots

Below are examples of the PROCHECK plots for a protein structure called here 1abc.

Each example contains a brief Description of the plot and the main Options that can be selected via the parameter file, procheck.prm (described in Customizing the PROCHECK plots).

By clicking on the plot example given you will get a larger version of it.

  • 2. Ramachandran plots by residue type

  • 8. Main-chain bond angle distributions
  • 9. RMS distances from planarity
