PhUbuntu: bootable Linux disk for running Phenix

PhUbuntu is a customized Ubuntu Linux "Live CD", capable of running on most Intel/AMD PCs with a DVD drive (and, potentially, support for booting from a USB flash drive). It includes full installations of Phenix and Coot, which are pre-configured in the shell environment. Because of the limitations of the booted system, this is not actually intended for people who want to use Phenix productively, but is used for tutorials and workshops where not all attendees have Mac or Linux laptops. It may also be useful for individuals who want to test Phenix on their own. (Note that while this disk image may be booted on some virtualization software, we recommend setting up a full Linux installation instead, assuming you have sufficient free space on your hard drive.)

To use this image, you will need to burn it to a DVD; most Windows PCs with DVD burners will include software to do this. (The image is too large to fit on a CD.) Some PCs will automatically boot from a DVD if present; in other cases, you will need to press a key when starting the computer to select the boot device. The exact procedure varies by manufacturer; consult the manual that came with your computer for details.

Currently, any data stored in the user home directory in the booted Linux environment will be lost when the computer is shut down. However, it should be possible to save data to a USB drive instead, which will enable you to transfer it to another system later. Additionally, networking support is very limited and our experience has been that most PCs will not have wireless ethernet connectivity while running this environment, but you may also use a USB key to bring any personal data that you wish to use in PHENIX. A number of example structures are provided with the PHENIX installation for running the various programs.
