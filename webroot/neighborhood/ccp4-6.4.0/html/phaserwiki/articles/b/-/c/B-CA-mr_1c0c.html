Tutorial 1

Search for 6 copies of Î²-Carbonic Anhydrase

Select New Project from the main menu
Select Set up tutorial data
Select the beta-Carbonic Anhydrase tutorial
Open the Molecular Replacement Tab
Open the Phaser-MR Tab
Load the Reflection Data file
Select the 3.5A dataset
Select the pdb file
Set the Variance to 61% id
Select the sequence for the composition
Set the composition to 6 copies
Search for 6 copies of the ensemble
The following "Additional Parameters" are set to reduce calculation time
Set the rotation peak selection to 65%
Turn off "Rescore rotation" and "Rescore translation"
Select "Fast" search method
Turn off the "Search deeper in the rotation list"

Run the tutorial, and check results.
