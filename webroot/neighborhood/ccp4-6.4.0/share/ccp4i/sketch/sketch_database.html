Web-Based Software to Use with the Monomer Library

Corina Structure Generation

http://www2.ccc.uni-erlangen.de/services/3d.html

A server which will generate a well-minimised 3D structure and return atom types which can be converted into CCP4 atom types.
Input either a SMILES string (use the Cactvs sketcher described below to generate these) or use a Java Applet molecular editor to sketch a structure.

Ask Corina to create a Sybyl file which can then be read in by the CCP4I Sketcher.
 

Cactvs 2D Structure Editor

http://www2.ccc.uni-erlangen.de/software/cactvs/index.html

From the same people as Corina - this is a downloadable package (executables currently available for most operating systems used in protein crystallography labs and easy to install).  The package includes a  powerful and easy to use sketcher which can be used to generate a SMILES string for input for Corina.
