CCP4i: Graphical User Interface
Interface to RasMol

The Interface allows the quick viewing of coordinates and maps through RasMol.

The external way of running RasMol (for now) is:
~mjh/programs/TkRasMol/rasmol8 (or rasmol24, if the graphics of the machine is up to it).

In addition to the menu options at the top of the RasMol screen, RasMol allows the execution of interactive commands. For this, type "t" in the RasMol screen, upon which a text bar appears at the bottom. With the focus (mouse) in that bar, commands may be typed in, which will be executed when Enter is pressed.

See program documentation: RasMol.
