GetWidget

NAME

GetWidget - return the list of widget ids for a given array element

SYNOPSIS

GetWidget arrayname variable_name

DESCRIPTION

For a parameter which is the element variable_name of array arrayname, return the id for the entry field, button or menu which controls the parameter value. In a few cases there may be more than one widget which controls a parameters value - then a list of all the widgets is returned.
