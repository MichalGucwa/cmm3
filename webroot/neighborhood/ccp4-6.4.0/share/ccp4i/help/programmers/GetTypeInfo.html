GetTypeInfo

NAME

GetTypeInfo - get type dependent information about an element in an array

SYNOPSIS

GetTypeInfo arrayname variable_name required_info

DESCRIPTION

For the element variable_name of the array arrayname, return information on required_info. Each parameter has a data type which is defined in the .def file (example data types are: _positivereal, _menu, _space_group). The information available on each data type is dependent on the class of the data type.

KEYWORDS

Data types
