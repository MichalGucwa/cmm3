WriteToLog

NAME

WriteToLog - Write text to log file.

SYNOPSIS

WriteToLog text

Optional Arguments:

-nohead
-nofoot

DESCRIPTION

Write the text text to the log file enclosed in a header which states that the output is from the CCP4i script and a 'foot' which marks the end of the output. If the -nohead or -nofoot arguments are used, the header or footer will not be written.
