The Fiddleit Example

There is a very simple example task fiddleit which is used to introduce the files required to define a task interface. Run the task interface stand-alone with the command:

> ccp4i -task fiddleit

The Demo Example

This is a more realistic example - for a program something like a simplified Refmac. Run the task interface stand-alone with the command:

> ccp4i -task demo

or, to make the demo accessible from the main window, edit the modules file $CCP4I_top/etc/modules and find the text string 'demo' near the top of the file. You should see two lines which are commented out with # characters at the beginning of the line. Delete the # characters. Run CCP4i:

> ccp4i

Select the Program List module from the contrast colour modules menubutton at the top left of the main window and then click on the Task Interface Demo button below it.
