GRAPH AND PLOT FILE MANIPULATIONS

In CCP4i, graph and plot file manipulations are done as part of other tasks, e.g. the program NPO is run as part of the Generate Patterson Map task in the Experimental Phasing module. To find out more, have a look in the CCP4i Program List 'module'.

Graphs produced in the .log file of many programs, can be displayed and manipulated with the Loggraph Utility.

CCP4 programs used in Graph and Plot file manipulations

BPLOT
Plot temperature factor by residue
HKLPLOT
Plot a precession photo from an HKL reflection file
HKLVIEW
Display zones of reciprocal space as pseudo-precession images (X-windows tool)
IPDISP
Display images from a variety of crystallographic sources (X-windows tool)
NPO
Molecule and map plotting
PLTDEV
Convert Plot84 meta-files to PostScript, Tektronix or HPGL
RASMOL
Molecular visualisation (X-windows tool)
STNET
Generate stereographic net plot for use with POLARRFN
XLOGGRAPH
Viewer for specially formatted 'log' files (X-windows tool)
XPLOT84DRIVER
Viewer for Plot84 meta-files (X-windows tool)

Non-CCP4 programs

Molscript
Display 3D structures, such as proteins, in both schematic and detailed representations
RasMol Home Page
Molecular visualisation program, superseded by:
Protein Explorer
RasMol-derivative for molecular visualisation
