Modules and Tasks

The Interface will run most frequently used programs from the CCP4 suite but is organised around the idea of tasks rather than programs. Usually one task corresponds to one program but sometimes more than one program may be required to perform a task or a program may be used in different tasks.

The tasks are grouped into modules according to the stage in the crystallographic process they are used in:

Data Reduction
Experimental Phasing
Molecular Replacement
Density Improvement
Model Building
Refinement
Structure Analysis
Validation & Deposition

There are four utility modules which contain tools which apply to the main types of data:

Map&Mask Utilities
Reflection Data Utilities
Coordinate Utilities
Graphics and Viewing Utilities

There is also a Program List module which has an alphabetic list of interfaced programs:

Program List
